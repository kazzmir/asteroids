# Asteroids

![screenshot](https://raw.github.com/kazzmir/asteroids/master/asteroids.png)

## Requirements

 * Scons
 * [r-tech1](https://github.com/kazzmir/r-tech1) library
 * libz

## Setup
```bash
# Basic build
$ scons

# With debugging symbols
$ export DEBUG=1
$ scons

# To run
$ ./asteroids
```
